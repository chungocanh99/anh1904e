Các phím tắt:
    1. ctrl + F: tìm kiếm trong file hiện tại
    2. ctrl + R: tìm kiếm và thay thế 1 cụm từ trong file kiếmện tại
    3. ctrl + shift + F: tìm kiếm trong toàn bộ project
    4. ctrl + shift + R: tìm kiếm và thay thế 1 chuỗi trong toàn bộ project
    5. shift + shift: tìm kiếm every where trong toàn bộ dự án
    6. shift + F6: rename file
