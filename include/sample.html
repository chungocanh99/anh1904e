-->lệnh include cho phép chúng ta import 1 file khác vào file hiện tại
    và tất cả những gì của file đó đều có thể sử dụng đc ở file đang nhúng

    cú pháp: include "path";
             include ('path');
    trong đó path là đường dẫn thư mục của file cần nhúng

    --> lệnh include_once cũng có tác dụng như include nhưng chỉ nhúng 1 file 1 lần duy nhất trong 1 file
    nghĩa là khi nhúng 1 file 2 lần trong 1 file thì ctr sẽ nhận khai báo 1 mà không báo lỗi gì cả

    --> lệnh require giống include
    --> lệnh require_once giống hàm include_once
