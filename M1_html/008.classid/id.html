Thuộc tính id trong html

    - id trong html nó giống như số chứng minh thư của 1 người
    nó là duy nhất trong html, và k đc phép trùng giữa các thẻ
    id phải viết liền, không có dấu, không có các ký tự đặc biệt
    tốt nhất nên có các chữ cái và các chữ số
    - class được sử dụng để gom nhóm giữa các thẻ html có điểm chung
    1 thẻ html có 1 hoặc nhiều class khác nhau
    các class thì cần phân tách = dấu cách ví dụ: mienbac top1
    quy tắc viết tên class viết liền, k dấu, không có ký tự đặc biệt
    - MỤC ĐÍCH của thuộc tính class và id là để kết hợp với các ngôn ngữ như css và js
    để thực hiện các tác vụ trên trang web
    - trong css thì cọn class bằng dấu . ở đầu, ví dụ: .mienbac
    - trong css để chọn id thì viết #id, ví dụ #hanoi
